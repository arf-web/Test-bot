# Twitch Informer

Ещё один учебный проект: телеграм бот, спрашивающий у Twitch'a какие каналы он-лайн, сколько на них зрителей и как давно идёт стрим. Каналы перед этим надо "добавить для отслеживания" в боте.
Данные сохраняются/выгружаются из json.
- more.py раз в минуту узнаёт, какие каналы активны, из добавленых для отслеживания всеми пользователями и выгружает в live_channels.json, а заодно хранит импортируемые функции и шаблоны для simple_twitch.py
- simple_twitch.py обрабатывает команды телеграм бота. 

Для работы с Telegram используется библиотека pyTelegramBotAPI; для Twitch - python-twitch-client
